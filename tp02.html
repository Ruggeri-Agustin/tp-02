(1.1) El constructor URL() devuelve un objeto URL recién creado que representa la URL definida por los parámetros.

(1.2) Una URL absoluta es una dirección web completa que incluye el protocolo, el nombre de dominio y la ruta completa del archivo o recurso.

(1.3) Las URL relativas son estructuras de URL que contienen la ruta de ubicación, omitiendo el nombre del sitio web

(1.4) Una dirección URL absoluta contiene toda la información necesaria para localizar un recurso.
Una dirección URL relativa localiza un recurso mediante una dirección URL absoluta como punto de partida.
